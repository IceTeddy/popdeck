# PopDeck

PopDeck is a native macOS pop-up launcher prototype built with SwiftUI and AppKit.

## Current Prototype

- Menu bar companion app with no Dock icon.
- Global shortcut: Command + backtick.
- Cursor-centered circular launcher panel.
- Starter actions for Finder, Safari, Terminal, System Settings, Downloads, and ChatGPT.
- Basic Settings scene for prototype status.

## Run

```bash
swift run
```

In Xcode, open this folder as a Swift package, select the `PopDeck` executable, and run it.

## Notes

This project is written from scratch. Orbit is used only as product and interaction reference.

# PopDeck v0.1.1

PopDeck v0.1.1 是一个自动更新验证版本。

## 更新内容

- 修复 GitHub Release 下载包的 app bundle 封装和签名问题。
- 为 Sparkle 自动更新准备新的发布包。
- 更新 appcast，让旧版本可以检测到新版本。

## 安装与更新

如果你已经安装 v0.1.0，可以在 PopDeck 菜单中选择“检查更新...”。

如果你是首次安装，也可以直接下载 `PopDeck-0.1.1.zip`，解压后移动到“应用程序”文件夹。

---

PopDeck v0.1.1 is an automatic update validation release.

## Changes

- Fix the GitHub Release app bundle packaging and signing issue.
- Prepare the new Sparkle update archive.
- Update the appcast so older versions can detect this release.

## Install Or Update

If you already installed v0.1.0, choose "Check for Updates..." from the PopDeck menu.

For a fresh install, download `PopDeck-0.1.1.zip`, unzip it, and move it to the Applications folder.

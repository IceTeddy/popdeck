# PopDeck v0.1.4

PopDeck v0.1.4 修复 Hub 连续呼出体验，并加入首次启动权限引导。

## 更新内容

- 修复连续关闭并重新呼出 Hub 后，鼠标 hover 效果可能失效的问题。
- 在设置页加入系统权限状态卡，显示辅助功能和输入监听是否已允许。
- 首次启动时，如果缺少关键系统权限，会显示一次轻量提示并引导打开系统设置。
- 手动下载和自动更新统一使用 DMG。

## 安装与更新

如果你已经安装旧版本，可以在 PopDeck 菜单中选择“检查更新...”。

如果你是首次安装，也可以直接下载 `PopDeck-0.1.4.dmg`，打开后将 `PopDeck.app` 拖到“应用程序”文件夹。

---

PopDeck v0.1.4 fixes repeated hub presentation behavior and adds first-launch permission guidance.

## Changes

- Fix a case where hover effects could stop working after repeatedly closing and reopening the hub without moving the pointer.
- Add a system permissions status card to Settings for Accessibility and Input Monitoring.
- Show a lightweight first-launch notice when required system permissions are missing.
- Use the DMG for both manual downloads and automatic updates.

## Install Or Update

If you already installed an older version, choose "Check for Updates..." from the PopDeck menu.

For a fresh install, download `PopDeck-0.1.4.dmg`, open it, and drag `PopDeck.app` to the Applications folder.

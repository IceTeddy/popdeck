# PopDeck v0.1.2

PopDeck v0.1.2 修复 Sparkle 更新弹窗没有跟随 PopDeck 语言设置的问题。

## 更新内容

- 为 app bundle 声明英文和中文本地化。
- 允许 Sparkle 使用 framework 内置的中文资源。
- 将中文设置映射到 Sparkle 实际提供的 `zh_CN` 本地化。

## 安装与更新

如果你已经安装旧版本，可以在 PopDeck 菜单中选择“检查更新...”。

如果你是首次安装，也可以直接下载 `PopDeck-0.1.2.zip`，解压后移动到“应用程序”文件夹。

---

PopDeck v0.1.2 fixes Sparkle update dialogs not following PopDeck's language setting.

## Changes

- Declare English and Chinese localizations in the app bundle.
- Allow Sparkle to use its built-in Chinese resources.
- Map the Chinese setting to Sparkle's available `zh_CN` localization.

## Install Or Update

If you already installed an older version, choose "Check for Updates..." from the PopDeck menu.

For a fresh install, download `PopDeck-0.1.2.zip`, unzip it, and move it to the Applications folder.

# PopDeck v0.1.0

PopDeck v0.1.0 is the first early preview release.

## Highlights

- Menu bar app with no Dock icon.
- Cursor-centered launcher panel.
- Configurable global shortcut.
- Drag, reorder, remove, and restore launcher items.
- Add apps, folders, files, and URLs.
- Optional launch at login.
- Chinese and English interface strings.

## Requirements

- macOS 14 or later.

## Install And Open

1. Download `PopDeck-0.1.0.zip`.
2. Unzip it and move `PopDeck.app` to the Applications folder.
3. Open PopDeck.

This build is currently unsigned. If macOS blocks the app:

- Control-click `PopDeck.app`, choose Open, then choose Open again in the confirmation dialog.
- Or open System Settings, go to Privacy & Security, find the blocked PopDeck message, and choose Open Anyway.

## Checksums

The SHA256 checksum is provided as a release asset:

```text
PopDeck-0.1.0.zip.sha256
```

## Known Limitations

- Automatic updates are not integrated yet.
- Developer ID signing and notarization are not enabled yet.
- This is an early preview build and behavior may change quickly.

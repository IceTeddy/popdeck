PopDeck

PopDeck 是一个轻量的 macOS 菜单栏启动器。按下快捷键，在鼠标位置打开应用、文件夹、文件和网页。

v0.1.5 · macOS 14+ · 开源 · 支持自动更新

Features

功能不多，但要顺手

鼠标位置呼出

Hub 会出现在指针位置，减少视线和鼠标来回移动，适合频繁切换应用和打开资料。

拖拽整理入口

应用、文件夹、文件和 URL 都可以加入。拖动排序、移除和恢复默认布局都在设置里完成。

安静常驻菜单栏

没有 Dock 图标干扰，支持开机启动。需要用时呼出，不需要时安静待在菜单栏。

应用内自动更新

新版本发布后，PopDeck 可以直接检查、下载、替换并重启，不需要反复去 GitHub 找安装包。

Help

首次打开遇到拦截？

当前版本尚未接入 Developer ID 签名。如果 macOS 提示无法验证开发者，可以按住 Control 点击 PopDeck.app，选择“打开”，并在确认弹窗中再次选择“打开”。

如果系统只显示“无法打开”或“已阻止使用”，请打开“系统设置” → “隐私与安全性”，在安全性提示中找到 PopDeck，然后点击“仍要打开”。这个按钮表示你确认要信任并打开刚下载的这个 App。

如果快捷键或鼠标呼出不可用，请在系统设置中为 PopDeck 授予“辅助功能”和“输入监听”权限。
